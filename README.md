# 🚀 Gestão de Clientes — Aplicação Full Stack

Aplicação completa de **Gestão de Clientes**, construída com **Angular** no frontend e **Node.js + Express + Prisma** no backend, seguindo boas práticas de arquitetura, modularidade e Clean Code.

# <h4>🚀 Deploy = https://gestao-clientes-one.vercel.app/</h4>


---

## 🧠 Visão Geral

Sistema web para gerenciamento de clientes com:

- 🔐 Autenticação JWT
- 👥 CRUD completo de clientes
- 📊 Dashboard com estatísticas
- 🔎 Filtros dinâmicos
- 🏗 Arquitetura organizada e escalável

Projeto desenvolvido com foco em **boas práticas de arquitetura**, **separação de responsabilidades** e **estrutura profissional para produção**.

---


## 🔹 Frontend

- Angular  
- RxJS  
- Angular Router  
- Lazy Loading  
- HTTP Interceptor  
- Reactive Forms  
- Estrutura modular (Core / Shared / Features)  

---




# 🛠️ Tecnologias Utilizadas

## 🔹 Backend

- Node.js  
- Express  
- TypeScript  
- Prisma ORM  
- PostgreSQL  
- JWT (JSON Web Token)  

Arquitetura: **Modular Monolítica**

---



# 🏛️ Arquitetura Modular Monolítica

O backend foi estruturado seguindo o padrão **Modular Monolith**:

- Aplicação única (um único deploy)
- Separação clara por módulos de domínio
- Cada módulo contém sua própria responsabilidade
- Escalável sem necessidade imediata de microserviços
